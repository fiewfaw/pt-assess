GitHub cloud sync

เก็บข้อมูลคนไข้ใน private GitHub repo ของคุณ → คอม+มือถือเห็นข้อมูลเดียวกัน + audit trail (git history) + backup อัตโนมัติ

📖 ขั้นตอนการ setup (5 นาที)
  1. ไป github.com/new สร้าง repo ชื่อ pt-data
  2. เลือก Private · ไม่ต้อง init README → Create
  3. ไป settings/tokens (fine-grained) → Generate new token
  4. Repository access: Only select repositories → เลือก pt-data
  5. Permissions → Repository → Contents: Read and write
  6. Copy PAT (ขึ้นต้น github_pat_...) แล้วใส่ในช่องด้านล่าง

📤 Export ข้อมูล (สำรอง / ส่งให้ Claude)

ดึงคนไข้+visit ทั้งหมดในเครื่องนี้ออกมาเป็น JSON (ไม่รวม PAT) — ใช้ตอน sync ไม่ได้
